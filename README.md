# Dyna Beacon

Wealth Beacon is a greenfield presale demo platform for mass-affluent and affluent RM teams.

## Local Setup

```bash
npm install
npm run check-data
npm run dev
```

Open `http://localhost:3000/login` and choose one of the demo RM accounts. The selected account is stored locally and controls customer visibility.

Default mode is `AI_MODE=demo`. Live Agent Studio credentials must stay in local environment variables and must not be committed.

## Project Milestones

- Phase 0: kickoff docs and module catalog.
- Phase 1: Next.js foundation, Dyna shell, repo layer, AI output/audit skeleton.
- Phase 2: local JSON seed data and validation.
- Phase 3+: RM workspace, AI copilot, communication flow, governance, deployment.
